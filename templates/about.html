Riddle Me This is a simple game written in Python 3 using HTML, CSS, Javascript and the Flask Framework to bring it all together.

The point of this game is to guess the answer to a riddle presented on screen and if guessed right, points will be added to your total score.

If your score is high enough, you will get put onto the highscore board.
